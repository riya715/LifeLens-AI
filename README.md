# LifeLens AI 🚀

LifeLens AI is an AI-powered journal analysis application built using Flask and TextBlob.

## Features

- Sentiment Analysis
- Mood Detection
- Productivity Tracking
- Mood Score
- AI Insights
- Word Count
- Reading Time Estimation
- Modern Responsive UI

## Tech Stack

- Python
- Flask
- TextBlob
- HTML
- CSS
- Git & GitHub

## How to Run

```bash
pip install flask textblob
python backend/app.py
```

Open:

http://127.0.0.1:5000
